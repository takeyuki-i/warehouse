# アプリーケーション名
個人倉庫管理システム

# アプリケーション概要
現在、企業に所属し働くのではなく、副業などの個人で事業を行う肩が増えてきています。
今後その傾向が強くなっていく中で、個人事業として売買を行っていくにあたり、売買品目が増えていくと
管理を行っていくのが大変です。
その為、個人または小規模企業が、EXCELなどの表計算ソフトを使用せずに一括で在庫管理をできるシステムを
作成致しました。
これにより、出荷すべき商品の優先順位・使用期限や保管場所、数量などを一元で管理できる事を目指し作成をしております。

# 利用方法
1.ユーザーの登録
2.商品の基本情報登録
3.商品毎のステータス（数量・保管場所・使用期限など）を登録
4.商品出庫の条件検索を行い出荷順に表示させる。
5.モーダルウインドウによる使用期限の通知

# 目指した課題解決
1.複数の商品管理が一元で出来る。
2.在庫数の管理が可能
3.期日の管理を行わなくてよくなる
4.出荷に関して出庫管理の手間が不要になる。
5.使用期限のある商品は設定した期日いアラームが出るようにする。
